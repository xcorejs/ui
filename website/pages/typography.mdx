Typography is block based component for texts a headings.

<Typography variant="h1">
  Heading
</Typography>

```jsx
import { Typograhy } from "@xcorejs/ui";

<Typography variant="h1">
  Heading1
</Typography>
```

## Aliases

For better usage, you can use aliases.

<Typography variant="h1">Heading 1</Typography>

```jsx
import { Typograhy } from "@xcorejs/ui";

<Heading1>Heading 1</Heading1>
or
<Typography variant="h1">Heading 1</Typography>
```


## Variants

Default: *p*

<Typography variant="p">Paragraph</Typography>
<Typography variant="lead">Lead text</Typography>
